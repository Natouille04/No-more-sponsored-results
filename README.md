# No More Sponsored Results

## Installation
1. Clone the repository to your computer.  
2. Open the [**Extensions**](chrome://extensions/) page in Chrome.  
3. Enable **Developer Mode** if it's not already enabled.  
4. Click **"Load unpacked"** and select the project folder.  
You're all set â€” the extension is installed!

## How to Use
Use Google as you normally would.  
The extension will automatically remove sponsored results from your searches.
